ARRAY:
-----
<script>
	var velocidades = []
	velocidades[0] = 10.1;
	velocidades[1] = 10;
</script>

Tambien podría dejar de indicar la posicion
<script>
	var velocidades = []
	velocidades[] = 10.1;
	velocidades[] = 10;
</script>
Mas sencillo aun:
<script>
	var velocidades = [10, 10.1, 9, 9.8, 10];
</script>

		var a = 4
		switch(a){
			case 1: console.log("a es 1");break

			case 3: console.log("a es 3");break
			default: console.log("Ni uno, ni dos ni tres");break

		}
	</script>
